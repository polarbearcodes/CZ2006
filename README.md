# CZ2006
Reddit for sadness

# How to run on local machine:
**Step 1**: Clone this repository <br />
**Step 2**: Open this repository and 2 terminals <br />
**Step 3**: On the first terminal, write command : <br />
   ```
   cd server
   yarn install
   yarn start
   ```
**Step 4**: On the second terminal, write command:<br />
   ```
   cd client
   yarn install
   yarn dev
   ```
